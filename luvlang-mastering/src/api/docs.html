Authentication

API Key Required

All API requests require an API key sent in the header:

X-API-Key: lm_your_api_key_here

Get your API key from your account dashboard after subscribing to a plan.

API Endpoints

POST /api/v1/analyze

Analyze audio file for LUFS levels, true peak, dynamic range, and more.

Request Body
multipart/form-data
File Field
audio
Returns
LUFS, True Peak, Dynamic Range
POST /api/v1/master

Apply mastering chain to audio with custom settings or presets.

Request Body
multipart/form-data
Fields
audio, settings (JSON), preset
Returns
Processed audio (base64)
POST /api/v1/master/auto Pro+

AI-powered automatic mastering with optional reference track matching.

Request Body
multipart/form-data
Fields
audio, reference (optional), targetPlatform
Returns
Auto-mastered audio + analysis
GET /api/v1/presets

Get list of all available mastering presets.

Query Params
category (optional)
Categories
music, podcast, content, custom
POST /api/v1/export

Export processed audio in specified format and quality.

Formats
WAV, MP3, FLAC, AIFF
Bit Depths
16, 24, 32 bit
Sample Rates
44.1k, 48k, 96k, 192k
POST /api/v1/batch Pro+

Process multiple audio files in a single request.

Max Files
10 per request
Fields
files[], settings, preset
GET /api/v1/user

Get current user account information and subscription tier.

GET /api/v1/usage

Get API usage statistics and remaining quota for current billing period.

Quick Example

Analyze an audio file using cURL:

curl -X POST /api/v1/analyze \ -H "X-API-Key: lm_your_api_key" \ -F "audio=@your-track.wav"

Response:

{ "success": true, "data": { "filename": "your-track.wav", "lufs": { "integrated": -14.2, "shortTerm": -13.5, "momentary": -12.1 }, "truePeak": -0.3, "dynamicRange": 10.5 } }
